# Healthy Recipes Finder

Ce projet est une application web conçue pour aider les utilisateurs à trouver des recettes saines facilement. En utilisant l'API Spoonacular, cette application permet aux utilisateurs de rechercher des recettes basées sur des critères spécifiques tels que le type de cuisine ou le nombre de recettes souhaitées. Ce projet illustre l'utilisation d'Express.js pour créer des routes API et de node-fetch pour interagir avec des services web externes.

## Fonctionnalités

* **Recherche de Recettes** : Les utilisateurs peuvent rechercher des recettes saines en spécifiant le type et le nombre de recettes.

* **Affichage des Résultats** : Les résultats de la recherche sont affichés sous forme de liste, incluant le titre et l'identifiant de chaque recette.
* **Gestion des Erreurs** : L'application gère les erreurs lors des requêtes à l'API Spoonacular, informant l'utilisateur en cas de problème.

## Technologies Utilisées

* **Node.js et Express** : Pour la création du serveur backend.
* **dotenv** : Pour la gestion des variables d'environnement.
* **node-fetch** : Pour effectuer des requêtes HTTP à l'API Spoonacular.
